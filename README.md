# Gamrin Boat Dashboard

#### This project is desgined to be a dashboard with some advanced features that uses a Rasberry PI and a Garmin Echomap 53CV. Displays real-time depth, GPS, speed, water temperature, and sonar trace on a touchscreen display. Runs completely offline — no internet or WiFi needed on the water.
